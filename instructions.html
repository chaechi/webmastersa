Restaurant Template User Instructions

1. Introduction

Welcome to the Multipurpose One Page Template! This template is designed to help you create a professional and visually appealing website using Bootstrap 5.3. It features a responsive design, a range of customization options, and animations to make your site stand out.

2. Installation

Follow these steps to install and set up your website template:

  1. Unzip the downloaded file.
  2. Open the folder and locate the index.html file. This is the main file for your template.
  3. Ensure you have the required files and folders, including css/, js/, and resources/ directories.
  4. Upload the contents to your web server or local development environment.
  5. Open index.html in your browser to view the template.

Note: Ensure you have a modern web browser to view and test the template.

3. Customization

To customize the template, follow these instructions:

3.1. Changing Colors

Update the color palette by modifying the CSS files located in the css/ folder. The primary color used is #e65c4e. You can change it in the styles.css file.

/* Change primary color */
                {
                    --primary-color: #e65c4e;
                    }

3.2. Updating Text Content

Edit the text content directly in the index.html file. For sections such as the menu, testimonials, and pricing, look for specific <section> tags and modify the content accordingly.

3.3. Modifying Images

Replace placeholder images in the images/ directory with your own images. Update image paths in the HTML files if necessary.

3.4. Responsive Design

The template is built using Bootstrap 5.3 and is responsive by default. To ensure responsiveness, use Bootstrap’s grid system and classes as described in the Bootstrap Grid Documentation.

4. Adding Custom Features

To add custom features or functionalities:

  1. Refer to the js/ folder for existing JavaScript files. You can add your custom scripts here.
  2. Update the index.html to include new sections or features. Use Bootstrap components for consistency.
  3. Ensure any new CSS styles are added to the styles.css file to maintain a consistent design.

Note: For advanced customizations, refer to the Bootstrap Customization Documentation.

5. Troubleshooting

If you encounter any issues:

  • Ensure that all file paths are correct and that no files are missing.
  • Check the browser console for any JavaScript errors and resolve them as needed.
  • Clear your browser cache and refresh the page to see changes.

Dark/Light Mode Toggle

The navbar includes a toggle switch for dark and light modes:

            <input type="checkbox" class="checkbox" id="checkbox">

                <i class="fas fa-moon"></i>
                <i class="fas fa-sun"></i>
                <span class="ball"></span>
            </label>
            <label class="change-mode-label" id="change-mode-label-id">Change Mode</label>

This switch is styled using Font Awesome icons. You can customize the functionality by updating the JavaScript file where the dark/light mode logic is implemented.

                        document.addEventListener('DOMContentLoaded', function () {
                            // Get the checkbox and html element
                            const checkbox = document.getElementById('checkbox');
                            const label = document.getElementById('change-mode-label-id');
                            
                            const htmlElement = document.documentElement;
                        
                            // Function to toggle theme based on checkbox state
                            function toggleTheme() {
                                if (checkbox.checked) {
                                    htmlElement.setAttribute('data-bs-theme', 'dark');
                                    label.textContent = 'Dark Mode'; // Change text for dark mode
                                } else {
                                    htmlElement.removeAttribute('data-bs-theme');
                                   
                                    label.textContent = 'Light Mode';
                                }
                            }
                        
                            // Initialize theme based on the current checkbox state
                            toggleTheme();
                        
                            // Add event listener for checkbox change
                            checkbox.addEventListener('change', function() {
                                toggleTheme();
                            });
                        });

JavaScript Functionality

The following JavaScript code is used to create a scrolling effect on the overlay text:

        <script>
        document.addEventListener("DOMContentLoaded", () => {
            const mainText1 = document.querySelector(".display-3");
        
            document.addEventListener("scroll", function () {
                let value = window.scrollY;
                mainText1.style.marginTop = value + "px";
            })
        });
        </script>

Make sure to include this script in your script.js file. This script selects the element with the class .display-3 and adjusts its margin based on the scroll position, creating a parallax scrolling effect.

        <h2 class="marquee-text">Your custom text here! •</h2>
        <h2 class="marquee-text">Your custom text here! •</h2>

Ensure the text in both <h2> elements is the same to create a seamless scrolling effect.

Adjusting the Animation Speed

The speed of the scrolling text can be adjusted by changing the duration in the animation property:

        .b-section-marquee-box h2 {
            animation: b-text-scroll 35s linear infinite;
        }

Decrease the duration value to make the text scroll faster, or increase it to make it scroll slower.

The font size of the marquee text is adjusted for smaller screens using a media query:

        @media (max-width: 600px) {
            .b-section-marquee-box h2 {

        }

You can modify the font size or add more styles within the media query to improve the appearance on smaller devices.

JavaScript Functionality

The following JavaScript prevents the card flip effect during scrolling to avoid undesired card flips: 


        document.addEventListener("DOMContentLoaded", () => {
            const cards = document.querySelectorAll(".chefSection .card");
            if (cards.length > 0) {
                function handleScroll() {
                    cards.forEach((card) => {
                        card.classList.add("no-flip");
                    });
                    setTimeout(() => {
                        cards.forEach((card) => {
                            card.classList.remove("no-flip");
                        });
                    }, 500);
                }
        
                document.addEventListener("scroll", handleScroll);
            }
        });

The following JavaScript code animates the skill bars when they come into view:


        function animateCounters(entries, observer) {
            entries.forEach(entry => {
                if (entry.isIntersecting) {
                    // Perform animations
                    
                    const skillSection = document.querySelector("#skills-section");
                    const circlesSection = document.querySelector("#section4");
        
                
                    if (skillSection) {
                        // Skill bar animations
                        const skills = document.querySelectorAll('.skill');
                        skills.forEach(skill => {
                            const percent = skill.getAttribute('data-percent');
                            const skillBarFill = skill.querySelector('.skill-bar-fill');
                            skillBarFill.style.width = percent + '%';
                        });
                    }
        
                    if (circlesSection) {
                        // Circular percentage bars
                        const circles = document.querySelectorAll('.circle');
                        circles.forEach(circle => {
                            const percent = circle.querySelector('.inner-circle').getAttribute('data-percent');
                            circle.style.background = `conic-gradient(from 0deg, #ff5e00, #00ffff ${percent}%, #333 ${percent}%)`;
                        });
                    }
        
                    // Unobserve after animation is triggered
                    observer.unobserve(entry.target);
                }
            });
        }
        
        let observerOptions = { threshold: 0.5 };
        let observer = new IntersectionObserver(animateCounters, observerOptions);
        
        let sections = document.querySelectorAll(".stats-section, #skills-section, #section4");
        sections.forEach(section => observer.observe(section));

  • The section title "What Our Clients Say" is inside an <h1> element with the class text-center mb-4 for center alignment and bottom margin.

Carousel:

  • A <div> with the ID reviewsCarousel and the class carousel slide is used for the carousel.
  • It includes attributes data-bs-ride="carousel" and data-bs-interval="5000" to control the carousel behavior.

Carousel Items:

  • Each carousel item is wrapped in a <div> with the class carousel-item. The first item also includes the class active.
  • Inside each carousel item, there is a row with the class row-cols-1 row-cols-md-3 g-4 to handle column layout and gaps.

Testimonial Cards:

  • Each testimonial is contained in a <div> with the class col position-relative.
  • The card itself uses the class testimonial-card p-4 bg-white shadow-sm rounded text-center for padding, background, shadow, rounded corners, and center alignment.

Customization

Adding/Removing Testimonials:

  • To add a testimonial, include a new <div> with the class carousel-item inside the carousel-inner div and follow the existing structure.
  • To remove a testimonial, delete the corresponding <div> with the class carousel-item.

Changing Images, Text, and Names:

  • Modify the <img> element's src attribute to change the image.
  • Adjust the content inside the <p> elements with the classes testimonial-text and testimonial-position to change the text and position.
  • Modify the content inside the <h5> element to change the name.
